Install Instructions
1- Install QGIS in https://qgis.org/download/ . OBS: choose 3.40 version LTR
2- Dependencies: Orfeo Tool Box 
 * Make download in https://www.orfeo-toolbox.org/download/. Choose OS and unzip in C:
3- To run plugin, unzip in C:\Users\name_user\AppData\Roaming\QGIS\QGIS3\profiles\default\python\plugins\urban_change_aid
4- In QGIS: 1- Go to options> processing > provider> OTB. In OTB set OTB application folder to: C:/otb910/lib/otb/applications and OTB folder C:/otb910
       2- In plugins, search by Plugin Reloader and install
5- To load Urban_change_aid Plugin, reload with Plugin Reloader. If Ok, will appear in plugin's menu
